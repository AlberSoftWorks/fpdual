# 🌊 FP Lanzarote 2026/2027 · Planificación Oficial

Portal web informativo sobre la oferta de **Formación Profesional en Lanzarote** para el curso académico **2026/2027**.

La plataforma permite consultar centros educativos, ciclos formativos y familias profesionales disponibles en la isla de forma clara, visual e interactiva, ofreciendo una experiencia moderna, responsive y accesible para estudiantes, familias y profesionales de la educación.

---

## 📌 Características principales

- 🏫 Visualización de centros educativos de Lanzarote  
- 📚 Consulta de ciclos formativos por centro  
- 🎯 Filtros dinámicos por familias profesionales  
- 📱 Diseño responsive adaptable a móviles y tablets  
- 🗺 Integración con Google Maps  
- 🔗 Accesos directos a páginas oficiales de los centros  
- ✨ Interfaz moderna con Font Awesome y Google Fonts

## ⚙️ Funcionalidades JavaScript

El proyecto incluye un sistema de filtrado dinámico que permite:

- Mostrar únicamente las familias profesionales seleccionadas  
- Ocultar automáticamente los centros sin coincidencias  
- Resaltar el filtro activo  
- Navegación fluida entre secciones mediante anchors  
- Interacciones visuales adaptadas a dispositivos móviles
